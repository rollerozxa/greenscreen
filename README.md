# Minetest Greenscreen Mod
Mod that adds various brightly-coloured nodes intended to be used for chroma-keying.

## Colours:
| Colour  |   Hex   |
|:-------:|:-------:|
| Black   | #000000 |
| Grey:   | #888888 |
| White:  | #FFFFFF |
| Red:    | #FF0000 |
| Orange: | #FF8800 |
| Yellow: | #FFFF00 |
| Green:  | #00FF00 |
| Cyan:   | #00FFFF |
| Blue:   | #0000FF |
| Purple: | #FF00FF |
